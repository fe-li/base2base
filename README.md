# base2base
Matlab/Octave script that converts number systems (2..64) - total and floating point

# How to use?

base2base(a,b,c,d)
- a -> it's a number of any system
- b -> is's number system (2..64)
- c -> it's system to convert (example from 10 to 2, so write 2)
- d -> it's float point
